Proyecto Cloud – To-Do List

Esta es la página web del proyecto de lista de tareas desplegada en un entorno tipo nube:

  • Backend en Node.js + Express
  • Base de datos SQLite
  • Aplicación empacada en contenedores Docker
  • Balanceo de carga con HAProxy (VM Balanceador)
  • Servicios REST disponibles en /tasks

Puedes probar los servicios desde Postman o el navegador usando:
GET http://192.168.1.63/tasks (a través del balanceador)
GET http://192.168.1.62:3000/tasks (instancia directa)

* Esta página se sirve desde el mismo backend que expone la API, corriendo en Docker detrás de HAProxy.
